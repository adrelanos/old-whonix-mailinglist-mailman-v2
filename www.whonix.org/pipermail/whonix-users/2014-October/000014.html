   By now I wanna use Whonix in practice with physical separation.
I have an old PIII machine left for the Whonix gateway; however I
am not too keen on compiling the code for the gateway from source
myself because this is sure to involve some further security issues.
First of all I am a bit in doubt about the security of git and other tools
involved to do so. Secondly it will be more hard to spot  alterations
if I do not have an image or a repository to compare against (always
need to keep checksums around on an usb stick).
   Last but not least it would be far easier to deploy if I had either an
image of a running chroot-environment for such a gateway or even
better .deb-s or an entire repository for the additional files needed
to set up a Whonix gateway. It would be really nice if someone was ready
to supply the community with one of the proposed distribution methods as
long as we do not have just a normal live and/or install CD for
that purpose!
   Before having a live Medium just for a Whonix gateway it would of course
be highly desirable to have a burnable live iso that already contains
qemu/virtualbox, the gateway and the client virtual machine. For people
with a high need for security things get far more easy as soon as you can
check a whole iso for integrity rather than having to check indivdual
files/images. CDs, DVDs and even BDs are my presonally preferred media
when it comes to security issues as they are read only and you can take
them with you.
