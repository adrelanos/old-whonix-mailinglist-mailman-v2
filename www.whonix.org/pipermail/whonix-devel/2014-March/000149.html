Hi Olivia,

thank you for reporting this.

It could be either, (1) that snapshot.debian.org was having
difficulties or (2) a rare but in apt-cacher-ng.

In case of (1), that issue should be gone by now. Just tested that.

In case of (2), please try the following commands before building Whonix.

   sudo service apt-cacher-ng restart
   sudo apt-get update
   sudo apt-get autoremove
   sudo apt-get dist-upgrade
   sudo apt-get clean
   sudo apt-get autoclean
   sudo service apt-cacher-ng restart

(Doesn't hurt to run these commands anyway.)

Cheers,
Patrick
