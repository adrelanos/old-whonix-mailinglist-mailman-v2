Hi kaltheat,

there is https://www.whonix.org/wiki/Dev/Design-Detailed and
https://www.whonix.org/wiki/Design (and also
https://www.whonix.org/wiki/Dev/SourceCodeIntro) but not as up to
date, verbose and complete as you and I wished it were.

You probably still end up digging through the source. Maintaining such
documentation would be too much a burden since Whonix's code is
continuously improved therefore obsoleting the documentation. I am
adding lots of source code comments why which step is done and I am
also planing on splitting Whonix's code into smaller packages to make
the core parts to get one started easier to find.
