Hi,

just announcing, Fortasse did me another favor and installed:
http://www.mediawiki.org/wiki/Extension:Replace_Text

Old situation:
if we changed a title on one page or a page name, all links broke.

New situation:
We can now use https://whonix.org/wiki/Special:ReplaceText and it able
to rename page names if wanted, set up a redirect to the new page name
and change all internal links.
