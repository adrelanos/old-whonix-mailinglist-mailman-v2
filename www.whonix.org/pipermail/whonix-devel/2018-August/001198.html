> * Then delete the newly created sys-whonix / anon-whonix.
>
> * Clone sys-whonix-backup to sys-whonix.
>
> * Clone anon-whonix-backup to anon-whonix.
>
> * Finally delete superfluous sys-whonix-backup / anon-whonix-backup.
